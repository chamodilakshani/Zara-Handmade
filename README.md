# Zara HandMade 🌸

Welcome to the **Zara HandMade** project! This is a specialized e-commerce platform dedicated to showcasing and selling handcrafted flower bouquets, pots, and unique floral items.

## Features ✨
* **User-friendly Interface**: A clean, aesthetic, and warm design inspired by nature.
* **Product Catalog**: Easily browse through handmade bouquets and pots with category filtering.
* **Gift Messaging**: A special feature to add personalized notes to your floral gifts.
* **WhatsApp Ordering**: Quick and direct ordering via WhatsApp integration.

## Tech Stack 💻
* **Frontend**: React.js with a custom warm-toned aesthetic.
* **Backend**: Node.js and Mongoose.
* **Database**: MongoDB for storing product and user data.
* **Deployment**: Dockerized environment for easy setup.

## Getting Started 🚀
1. Clone this repository:
   `git clone https://github.com/chamodilakshani/Zara-Handmade.git`
2. Run the application using Docker:
   `docker compose up --build`
